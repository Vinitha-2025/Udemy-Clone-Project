# 🎓 Udemy-Clone-Project

A responsive front-end clone of the Udemy website built using HTML and CSS. This project recreates the look and feel of Udemy's homepage, featuring course listings, category navigation, recommended topics, and a fully responsive layout.


# 📖 About The Project

This project is a responsive clone of the Udemy homepage created to improve my front-end web development skills. The design includes a modern navigation bar, search functionality, course sections, topic recommendations, and a structured footer similar to the original Udemy platform. The project focuses on creating clean layouts, responsive design techniques, and reusable UI components using only HTML and CSS.


# ✨ Features

- Responsive Navigation Bar
- Search Input Section
- Course Categories Navigation
- Hero Banner Section
- Trending Courses Section
- Featured Courses Section
- Recommended Topics Section
- Course Cards with Pricing and Ratings
- Professional Footer Layout
- Mobile-Friendly Responsive Design


# 🛠️ Built With

- HTML5
- CSS3
- Flexbox
- Media Queries
- Font Awesome Icons
- Google Fonts (Poppins)


# 📱 Responsive Design

The website is optimized for:

- 💻 Desktop
- 📱 Mobile
- 📟 Tablet


# 🎯 Purpose

This project was built to improve my front-end web development skills by recreating a responsive version of the Udemy homepage. It helped me practice HTML page structure, CSS styling, Flexbox layouts, responsive design techniques, and creating modern user interface components. Through this project, I gained hands-on experience in building real-world website layouts and improving my understanding of web design principles.
